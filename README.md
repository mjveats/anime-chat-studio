# Anime Chat Studio 🎬

A web-based studio for generating viral texting stories optimized for YouTube Shorts and TikTok. 

---

## ⚡ How to Use

### 1. Build Your Cast
* Go to the **Library** tab to add characters, or go to the **Create** tab to add your own using an image URL.
* Click a character in the Library to add them to your **Active Cast**.
* **Select the POV:** Click the radio button next to a character's name in the Active Cast. Their messages will appear on the right side, while everyone else appears on the left.

### 2. Write the Script
Use the Script Editor to write your story with this syntax:
* **Normal Message:** `Name [Delay in seconds]: Message` *(e.g., Luffy [1.5]: I want meat!)*
* **Fast Reply:** Add `>>` at the start of a line. *(e.g., >> Sanji [0.5]: No!)*
* **System Actions:** `sys: Message` *(e.g., sys: Nami left the chat.)*
* **Reactions:** `react: 💀` *(Place immediately under the message)*
* **Images:** `Name [1]: img:YOUR_IMAGE_LINK`

### 3. Customize the Chat
* **Group Info:** Set the Group Name in the bottom right corner.
* **Background:** Paste an image URL in the "BG URL" box.
* **Custom Sounds:** Use the **Sounds** tab to upload your own MP3/WAV files for typing, sending, and receiving messages.

### 4. Launch and Record
1. Click the blue **🚀 Launch Output Window** button. 
2. **Toggle Safe Zone:** Click the Ruler icon at the top right of the pop-up to see the YouTube Shorts safe areas. Ensure your text isn't covered by the red boxes.
3. Open your screen recorder (like OBS) and record *only* the pop-up window.
4. Go back to the main studio screen and click **▶ PLAY SCRIPT**.
